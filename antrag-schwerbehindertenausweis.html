Antrag auf Feststellung einer Behinderung / Schwerbehindertenausweis

Zur Übersicht: Schwerbehindertenrecht

Mit diesem Formular können Sie Angaben für einen Antrag auf Feststellung des Grades der Behinderung (GdB) und ggf. von Merkzeichen zusammenstellen sowie die Ausstellung eines Schwerbehindertenausweises beim zuständigen Versorgungsamt vorbereiten. Das Tool unterstützt bei der Strukturierung Ihrer Informationen (z. B. Gesundheitsstörungen, Auswirkungen im Alltag, Behandler, Anlagen) und kann als Grundlage für den offiziellen Antrag oder als Begleitschreiben dienen. Hintergrundwissen und Hinweise zur Vorbereitung finden Sie hier: Informationen zu diesem Thema.

Hinweis zur Datenspeicherung: Die Funktionen „Eingaben speichern“ und „Gespeicherte Eingaben laden“ nutzen eine lokale Speicherung im Browser (gerät-/browserbezogen). Es findet dabei keine automatische Übermittlung Ihrer Formulardaten an Kassen-Lotse statt.

Hinweis zur inhaltlichen Prüfung: Dieses Tool stellt keine Rechtsberatung dar und ersetzt keine individuelle Beratung. Das erzeugte PDF ist ein Vorschlag auf Basis Ihrer Eingaben und sollte vor dem Einreichen von Ihnen geprüft und bei Bedarf ergänzt oder korrigiert werden. Bei Unsicherheiten kann eine anerkannte Beratungsstelle oder anwaltliche Unterstützung sinnvoll sein.

Technischer Hinweis: Um das PDF erfolgreich herunterzuladen, öffnen Sie diese Seite bitte in einem Internet-Browser wie Chrome, Firefox oder Safari. In der Google-App (z. B. über die Suchleiste auf dem Startbildschirm) funktioniert der Download leider nicht zuverlässig.

Redaktionell geprüft: 01.02.2026

1. Angaben zur Person, für die der Antrag gestellt wird
2. Antragsteller:in (falls abweichend von oben genannter Person)
3. Angaben zu den Gesundheitsstörungen

Bitte führen Sie hier alle Gesundheitsstörungen auf, die berücksichtigt werden sollen. Für die behördliche Prüfung sind eine möglichst klare Bezeichnung, ein ungefährer Beginn sowie passende Unterlagen (z. B. Arztbriefe/Befunde) hilfreich.

4. Behandelnde Ärzte, Krankenhäuser, Rehabilitationseinrichtungen

Bitte geben Sie Stellen an, die Ihre Gesundheitsstörungen behandeln oder behandelt haben (oft hilfreich: letzte 2–5 Jahre). Das Versorgungsamt kann dort ggf. Befundberichte anfordern. Entbinden Sie die Ärzte von der Schweigepflicht (siehe Anlagen).

5. Beantragte Merkzeichen (falls zutreffend)

Hier können Sie angeben, ob Sie die Feststellung bestimmter Merkzeichen beantragen. Eine kurze Begründung kann hilfreich sein; die Nachweise ergeben sich typischerweise aus Befunden/Arztberichten.

6. Frühere Anträge / Bestehender Ausweis

Wenn Sie eine Verschlimmerung beantragen, wird in der Praxis häufig eine erneute Bewertung der Gesundheitsstörungen vorgenommen. Die konkrete Vorgehensweise hängt vom Einzelfall und der zuständigen Behörde ab.
